# GameLog Project Status

## Overall Minimum Viable Project (MVP) Feature List

1. **Game Search & Database Integration** — Connect to IGDB or RAWG API for game data
2. **Game Logging** — Add games with status: Played / Playing / Want to Play
3. **Ratings & Reviews** — Rate games (1–10 or star system) with optional text review
4. **User Accounts & Authentication** — Sign up, log in, user profiles
5. **User Profile Page** — Public-facing page showing a user's logged games, ratings, and reviews
